    • How To Use Krackin

    Since Krackin utilises a chatbot as a means of controling its sub functions it takes one word commands to trigger them

      Input Commands

        • Password Checker : PasswordChecker + (password)
          • Calculator : maths + (equation)
            • IP retrival : GetMyIp
              • Phishing Detector : Phisher + (URL)
                • Network Scanner : net + (IP Range )
                  • Password creator : PasswordCreate
                    • File Encrypter : encrypt
                      • File Decrypter : decrypt
                        • File Deleter: delete history
